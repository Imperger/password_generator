## Template variables

| Variable | Description | Example Output |
|----------|------------|----------------|
| `{rnd}`  | Generates a random string based on current generator options. | `a8F3k2` |
| `{date}` | Current local date in `YYYY-MM-DD` format. | `2025-12-14` |
| `{year}` | Current year (4 digits). | `2025` |
| `{month}` | Current month (2 digits). | `12` |
| `{day}` | Current day of the month (2 digits). | `14` |
| `{time}` | Current local time in `HH:mm:ss` format. | `14:32:10` |
| `{ts}` | Current Unix timestamp (seconds since epoch). | `1734186730` |
